SortME

An open-source robotic image recognition platform.

Download as .zip Download as .tar.gz View on GitHub

Say hello to SortME, the computer vision robot!

The purpose of this project is to provide a platform that will be able to track a range of objects. It will run on the beaglebone black and will send data over serial to an arduino. Check the API for documention.
