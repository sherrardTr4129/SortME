SortME

An open-source robotic image recognition platform.

Download as .zip Download as .tar.gz View on GitHub

Say hello to SortME, the computer vision robot!

The purpose of this project is to provide a platform that will be able to track a range of objects. It also will (hopefully) serve as a guide for anyone interested in learning about computer vision, OpenCV, and robotics.
